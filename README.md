# WebAccess Compliance Checker

## Overview
WebAccess Compliance Checker is an AI-powered platform that helps developers identify, analyze, and fix web accessibility issues in websites and code repositories.
Users can submit either a live website URL or a GitHub repository link. The platform evaluates the project against predefined accessibility rules and WCAG guidelines, generates a detailed compliance report, and provides AI-generated code fixes for detected issues.
Developers can review suggested modifications, compare changes side-by-side, and choose to accept, reject, or undo fixes. Approved changes are automatically applied, making accessibility compliance faster and more efficient.
---
## Features
* Website accessibility auditing using URL analysis.
* GitHub repository accessibility scanning.
* WCAG-based compliance evaluation.
* Accessibility compliance score generation.
* Rule-by-rule issue breakdown.
* Severity classification of violations.
* AI-generated code remediation suggestions.
* Side-by-side code comparison.
* Accept, reject, or undo suggested fixes.
* Automatic application of approved changes.
* Re-evaluation and updated compliance reports.
---
## How It Works
1. Enter a Website URL or GitHub Repository URL.
2. The platform scans the website or source code.
3. Accessibility violations are identified.
4. A compliance score and detailed report are generated.
5. AI suggests accessibility improvements.
6. Users review proposed code changes.
7. Approved fixes are automatically applied.
8. The project is rescanned to measure improvement.
